# 🎉 Google DevFest Workshop - Chrome DevTools MCP Server

> 🌈 一个超级可爱的 Google DevFest Workshop 活动官网

## 📖 项目介绍

这是一个为 Google DevFest Workshop 活动创建的官方网站，主题是 **Chrome DevTools MCP Server**。网站采用超级可爱的设计风格，包含丰富的动画效果和表情符号。

## 🌟 网站特色

- 🎨 **超级可爱的设计风格** - 大量使用表情符号和渐变色彩
- ✨ **丰富的动画效果** - 浮动动画、发光效果、悬停动画
- 📱 **完全响应式设计** - 适配桌面、平板、手机等各种设备
- 🚀 **优秀的用户体验** - 清晰的导航、直观的布局、愉悦的交互

## 🚀 本地运行

1. 克隆仓库到本地：
```bash
git clone https://github.com/YOUR_USERNAME/devtools-mcp-website.git
cd devtools-mcp-website
```

2. 启动本地服务器：
```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080
```

3. 在浏览器中访问：http://localhost:8080
